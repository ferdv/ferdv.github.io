Vim: Replace With a Sequence of Same Characters

Mar 18th, 2015 1:48 pm

When substituting in Vim, one can use the result of evaluating a Vim expression as the replacement string. I found it useful to replace a string by a sequence of the same character. For (a somewhat contrived) example, say we want to hide words enclosed in [ ].
