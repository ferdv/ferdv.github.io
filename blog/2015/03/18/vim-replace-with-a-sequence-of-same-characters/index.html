Vim: Replace With a Sequence of Same Characters

Mar 18th, 2015 1:48 pm

When substituting in Vim, one can use the result of evaluating a Vim expression as the replacement string. I found it useful to replace a string by a sequence of the same character. For (a somewhat contrived) example, say we want to hide words enclosed in [ ].

Lorem ipsum [dolor] sit amet, consectetur [adipiscing] elit. Fusce sed
vestibulum eros. Aliquam et ipsum quis est [interdum] pellentesque.

which gives us:

Lorem ipsum _____ sit amet, consectetur __________ elit. Fusce sed
vestibulum eros. Aliquam et ipsum quis est ________ pellentesque.

\= allows to enter a Vim script expression, the value of which will be the substitution string. The function submatch returns the string matching the (sub-)pattern. Passing 0 as the parameter returns the match for the whole pattern. Greater values correspond to sub-patterns enclosed in \( \). These are normally referred to as \1, \2, etc., in the substitution. The remaining two functions are quite straightforward: strlen gets the length of a string and repeat repeats its first argument n-times, where n is the second argument.
